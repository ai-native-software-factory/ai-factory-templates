## 📐 方法论说明

本知识库采用 **TOGAF ADM** 分层架构 + **CPM 能力规划**：

| TOGAF 层 | 关注点 | 知识库目录 |
|---------|--------|---------|
| A. 架构愿景 | 战略、商业模式 | 01-战略层 |
| B. 业务架构 | 业务能力、流程 | 02-业务架构 |
| C. 数据架构 | 实体、关系、数据流 | 04-数据架构 |
| D. 应用架构 | 微服务、API | 03-应用架构（能力域 E~H） |
| E. 技术架构 | 基础设施、技术选型 | 05-技术架构 |
| H. 架构治理 | 规范、代码标准 | 06-运营规范 |

**CPM 能力域**:
- A=授权管理、B=学员管理、C=内容运营、D=订单计费、E=直播交付
- F=分销管理、G=测评评估、H=账号管理
